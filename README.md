# SKYCOLLECTOR# 
# SkyCollector ✈️

실시간 항공기 데이터를 기반으로 주변 하늘을 지나가는 비행기를 탐지하고 수집하는 항공기 수집형 웹 프로토타입입니다.

## 주요 기능

- OpenSky API 기반 실시간 항공기 데이터 수집
- 사용자 위치 기준 50km 이내 항공기 표시
- 가장 가까운 항공기 탐지
- 사용자 위치와 항공기 연결선 시각화
- 신규 항공기 자동 수집
- 수집 도감 생성
- 수집 개수, 최근 수집, 항공사 통계 표시
- Rare Aircraft 알림
- 레벨 및 XP 시스템
- 업적 시스템

## 사용 기술

- Python
- Folium
- OpenSky Network API
- HTML
- GitHub
